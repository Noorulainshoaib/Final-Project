#MERN Stack Project
Shopping World


Technologies Used
Frontend:

React.js
React Router
Axios (for API requests)

Backend:

Node.js
Express.js
MongoDB (Database)
Mongoose (ODM for MongoDB)

Development Tools:

VS Code (Code Editor)
Git (Version Control)
Postman (API Testing)

Backend Api 
1) User Api 
a)   POST /api/signup
b)   POST /api/login

2) Category Api
a)  get /api/get-all-categories
b)  post /api/create-category

3) Product Api
a) get /api/products
b) post /api/add-product
c) delete /api/delete-product
